COPIANDO ARQUIVOS - CP

COPIANDO TODOS OS ARQUIVOS COM EXTENSÃO .TXT PARA O DISK2
cp /home/rhy/*.txt /disk2

COPIANDO TODOS OS ARQUIVOS
cp /home/rhy/* /disk2

COPIANDO TODOS OS ARQUIVOS E DIRETÓRIOS RECURSIVAMENTE, PARAMETRO -r
cp -r /home/rhy/* /disk2

COPIANDO TODOS OS ARQUIVOS E DIRETÓRIOS RECURSIVAMENTE E COM INTERAÇÃO DO QUE ESTÁ SENDO COPIADO, PARAMETRO -i
cp -r -v /home/rhy/* /disk2

UTILIZAR O PARAMETRO -v PARA VERIFICAR O QUE ESTÁ SENDO COPIADO E IDEAL PARA ARQUIVOS GRANDES


MOVENDO E RENOMEANDO ARQUIVOS - MV

MOVENDO ARQUIVOS PARA O DISK2
mv /home/rhy/arquivo1.txt /disk2

RENOMEANDO NOME DO ARQUIVO
mv arquivo.txt arquivo-renomeado.txt


INICIANDO, VISUALIZANDO E ENCERRANDO UM PROCESSO

COMANDO PS MOSTRA APENAS OS PROCESSOS DO BASH E USUÁRIO ATUAL
ps

MOSTRAR TODOS OS PROCESSOS EM EXECUÇÃO (PARAMETRO A), QUAL USUÁRIO (PARAMETRO U), PROCESSOS EXECUTADOS FORA DO CONSOLE BASH (PARAMETRO X)
ps aux

MATAR PROCESSO, PODE SER PELO NOME DO PROCESSO OU PID
kill

MATAR TODOS OS PROCESSOS COM O NOME CHROME
killall chrome

VERIFICAR QUAIS USUÁRIOS ESTÃO LOGADOS ATUALMENTE
w

O COMANDO "WHO -A" VAI MOSTRAR O PID DOS USUÁRIOS
who -a

PARA MATAR A EXECUÇÃO DO USUÁRIO PELO PID
kill pid-usuario
